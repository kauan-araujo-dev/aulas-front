Trabalhando com Listas

Listas sâo estruturas para representação de conteúdo na forma de relação de itens

Lista não-ordenada

Relação de itens em que a sequencia não importa

Exemplo:

  • Teclado
  • Mouse
  • Monitor
  • Webcam

Lista ordenada

Relação de itens em que a sequencia importa

Exemplo:

  1. Andressa
  2. Bruno
  3. Carlos
  4. Kauan

Listas aninhadas (listas dentro de listas)

Podemos misturar listas de diferentes tipos e tambem colocar listas dentro de outras listas

Exemlos

  • Doces:
    • Brigadeiro
    • Torta de Limão
    • Paçoca
  • Salgados:
    • Coxinha
    • Pizza

Lista de Definição

Lista usada ára estruturas de itens baseadas em termo e descrição. Útil para conteúdos do tipo: glossários, FAQs, ou apresentação de dados com algum conceito etc.

Exemplos:

Processador
Processa e armazenda dados
Peça essencial do computador
Mémoria RAM
Espaço de armazenamento volátil
